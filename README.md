# 🎵 Neumorphic Music Player

A stylish and responsive **Neumorphic Music Player** built using **HTML, CSS, and JavaScript**.
This project features a modern UI design with smooth animations, playlist drawer, progress tracking, and interactive music controls.

## ✨ Features

* 🎧 Modern neumorphic UI design
* ▶️ Play / Pause functionality
* ⏭️ Next & Previous track controls
* 🔁 Repeat mode toggle
* 📂 Sliding playlist drawer
* 📱 Fully responsive layout
* 🎨 Smooth animations and transitions
* ⏱️ Interactive progress bar with seek support

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Google Fonts & Material Icons

## 📁 Project Structure

```bash
├── index.html
├── style.css
└── script.js
```

## 🚀 How to Run

1. Open `index.html` in your browser.
2. Enjoy the music player UI.

## 📌 Notes
* Uses online images and Spotify track links.
